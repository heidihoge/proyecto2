Se listan los productos y se puede agregar nuevos. Siga los siguientes pasos: 

 

Nuevo Producto :

• Complete de manera obligatoria los siguientes campos:
  1. Codigo: sera puesto por la escuela para control interno.

  2. Nombre.

  3. Descripción.

  4. Control de Stock.

  5. Precio de Venta : el precio final por el cual sera vendido el producto.

  6. Costo : precio de compra del producto.

  7. IVA : 5%, 10% o Excentas.

  8. Estado : debe estar activo para poder vender o inactivo si ya no se desea ver en el listado.

  9. Existencia: debe tener un numero mayor a cero dentro del sistema.

• Y podemos guardar el producto

• Complete de manera opcional el siguiente campo: Foto producto.
